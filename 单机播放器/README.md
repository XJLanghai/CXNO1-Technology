材料：>Arduino UNO *1、无源蜂鸣器 *1、公对公杜邦线 *3、母对母杜邦线 *3、串口线 *1、电源适配器 *1

规格（PCB板）：
>Arduino UNO单片机：长67.8mm 宽52.5mm

>无源蜂鸣器：长32.5mm 宽13.3mm

设计思路：
>使用5v和GND为蜂鸣器供电与接地
>在Arduino IDE软件中进行编程：
>将8号引脚设为信号输出，随着时间的变化不断改变调节输出信号的频率。
>蜂鸣器上的GND、I/O、VCC分别与Arduino UNO上的GND、8号引脚、5v连接。
